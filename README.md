# Finance Tale

Aplicativo mobile para gerenciamento de finanças pessoais desenvolvido com React Native, Expo e TypeScript.

## Visão Geral

O Finance Tale foi criado com o objetivo de oferecer uma experiência simples e intuitiva para controle financeiro, permitindo o gerenciamento de carteiras e despesas diretamente pelo dispositivo móvel.

O projeto está sendo desenvolvido como parte do meu processo de aprendizado e aperfeiçoamento em desenvolvimento mobile com React Native.

## Funcionalidades

- Cadastro de carteiras financeiras
- Registro de despesas
- Edição de gastos
- Exclusão de gastos
- Histórico financeiro
- Armazenamento local dos dados
- Interface otimizada para dispositivos móveis

## Tecnologias Utilizadas

- React Native
- Expo
- TypeScript
- Expo Router
- AsyncStorage

## Instalação

Clone o repositório:

```bash
git clone https://github.com/iT4lloEdw/finance-tale.git
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npx expo start
```
